# Nutrilink-MSU-FoodSecurity-01

## Overview

NutriLink is a Retrieval-Augmented Generation (RAG) proof-of-concept developed through the Mississippi Artificial Intelligence Innovation Hub in collaboration with the Mississippi Department of Human Services (MDHS). The project explores how generative AI and cloud-native services can improve access to food assistance and nutrition-related resources for Mississippi residents.

The PoC focuses on Hinds County, Mississippi, and demonstrates how natural-language querying can be combined with county-specific datasets, nutritional guidance frameworks, and AWS AI services to provide conversational assistance related to food insecurity and nutrition security.

The system uses Amazon Bedrock Knowledge Bases with the Nova 2 Lite foundation model to retrieve contextual information from uploaded datasets and generate responses to user questions.
