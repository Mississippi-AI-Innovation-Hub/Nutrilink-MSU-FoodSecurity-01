# Setup Instructions

These instructions describe the intended sandbox setup for the NutriLink PoC.

## Prerequisites

- AWS account or sandbox environment.
- Access to Amazon Bedrock.
- Access to Amazon Bedrock Knowledge Bases.
- Amazon S3 bucket for markdown documents.
- AWS Lambda.
- Amazon API Gateway.
- Basic HTML/JavaScript frontend.

Cleaned sample datasets were uploaded to the sandbox into S3 buckets, then referenced as knowledge bases by either the Sandbox or API calls.
